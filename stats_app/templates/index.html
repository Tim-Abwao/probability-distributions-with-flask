Probability Distribution Sampler

This is a simple web app to help explore some useful probability distributions, and showcase the functionality of some Python packages (Flask, SciPy, Pandas, Matplotlib and Seaborn, among others).

The idea is to enable anyone to:

  • get basic facts about a distribution
  • specificy parameters of choice
  • generate a downloadable sample, and
  • view graphs and summary statistics.
